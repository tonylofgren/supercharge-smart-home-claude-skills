description: Create ESPHome device configurations for ESP32/ESP8266
---

Activate the esphome skill for creating device firmware configurations.

**What this does:**
- Creates complete ESPHome YAML configs
- Supports 160+ components (sensors, displays, LEDs, etc.)
- Includes WiFi, API, OTA setup

**Usage:** Just describe what device you want after invoking this command.

Example: `/esphome` then "Create a temperature sensor with OLED display"
