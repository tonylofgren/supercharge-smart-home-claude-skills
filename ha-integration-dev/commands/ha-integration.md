description: Create Python custom integrations for Home Assistant (custom_components)
---

Activate the ha-integration skill for developing Python integrations.

**What this does:**
- Creates complete custom_components structure
- Implements config flows, coordinators, entities
- Follows Home Assistant Core patterns
- Prepares for HACS publishing

**Usage:** Just describe what integration you want after invoking this command.

Example: `/ha-integration` then "Create a custom integration for my cloud API"
