# Node-RED Flow Templates

Ready-to-import flow templates for common Home Assistant automations.

## How to Use

1. Copy the JSON from the template file
2. In Node-RED, go to Menu → Import
3. Paste the JSON
4. Click Import
5. Configure the server node and entity IDs
6. Deploy

## Important Notes

- **Server configuration**: All templates leave the `server` field empty. You must select your Home Assistant server after importing.
- **Entity IDs**: Templates use placeholder entity IDs like `binary_sensor.motion_CHANGE_ME`. Update these to match your actual entities.
- **Comment nodes**: Each template includes a comment node explaining required configuration.

## Template Structure

Each template follows this pattern:

```
┌──────────────────────────────────────────────────────────────┐
│  [comment: Configuration Instructions]                        │
│                                                              │
│  [trigger] ──> [logic/function] ──> [action]                 │
│                                                              │
│  [debug: For testing]                                        │
└──────────────────────────────────────────────────────────────┘
```
