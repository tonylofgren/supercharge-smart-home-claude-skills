# Node-RED Flow Templates

Ready-to-import JSON flows for common home automation scenarios.

## How to Use

1. Copy the JSON from the template file
2. In Node-RED: Menu → Import → Clipboard
3. Paste and click Import
4. Configure entity IDs for your setup
5. Deploy

## Customization

### Entity IDs

Each template uses placeholder entity IDs. Replace with your actual entities:

```json
// Before
"entityId": "binary_sensor.motion_living"

// After - your entity
"entityId": "binary_sensor.pir_sensor_living_room"
```

### Environment Variables

Some templates use environment variables for easy customization:

```
MOTION_TIMEOUT - Seconds before turning off (default: 300)
BRIGHTNESS_DAY - Daytime brightness % (default: 100)
BRIGHTNESS_NIGHT - Nighttime brightness % (default: 30)
```
